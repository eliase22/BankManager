# BankManager
Description
Application Java simple illustrant l'héritage avec différents types de comptes bancaires.

Classes
BankAccount (classe parent)
CheckingAccount
SavingsAccount
COD (Certificat de dépôt)

Fonctionnalités
Création de comptes
Accès aux attributs (get/set)
Affichage des informations

Exécution
Compiler : javac *.java
Lancer : java Main
